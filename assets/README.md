# War of Nations Assets

This folder contains all game assets for the War of Nations strategy game.

## Structure
- `sounds/` - Audio files (explosions, movements, etc.)
- `music/` - Background music tracks
- `images/` - 2D sprites and icons
- `models/` - 3D models for buildings and units
- `ui/` - UI elements and skins

These are placeholder files. Replace with actual assets for production.
